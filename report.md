## 2026-06-15 初版作成

### 概要
スマホ（iPhone / Android）のブラウザで使う、Bar用の1日売上記録アプリ。
サーバー不要・オフライン動作・データは端末内（localStorage）に自動保存。

### ファイル
- `index.html` … アプリ本体（HTML/CSS/JS すべて1ファイル完結）

### 実装した機能
- **会計タブ**：国名（ボタン選択＋自由入力）／人数／自動「○組目」表示／チャージ（ON-OFF可、人数×単価）／ドリンク追加・杯数調整／ドリンクごとの「ソーダ +100円」トグル／会計確定→本日一覧へ記録
- **ドリンクタブ**：初期登録（ビール800・梅酒800・ハブ酒1500）、名前/値段の追加・編集・削除、チャージ料金とソーダ追加料金の変更
- **現金タブ**：紙幣・硬貨（10000〜1円）の枚数入力→現金合計を自動集計
- **経費タブ**：内容＋金額の記録、経費合計
- **集計タブ**：本日の売上／来客（組数・人数）／チャージ合計／ドリンク売上／経費合計／売上−経費／現金実数／国別人数、集計テキストのコピー、営業終了リセット（ドリンク登録・料金設定は保持）

### バックアップ
- git 初期化済み。初版コミット：`c2d26f6 バックアップ: Bar会計ノート 初版`

### 検証
- 埋め込みJSの構文チェック：OK（`node --check`）
- 金額計算ロジックのテスト：全項目期待値一致
  - 会計合計（チャージ2000＋ビール1600＋梅酒ソーダ900）= ¥4,500
  - 売上合計 ¥6,000／ドリンク売上 ¥4,000／売上−経費 ¥5,500
  - 現金集計 ¥36,734／国別 アメリカ2名・日本1名
- ※プレビュー環境が別プロジェクト(Next.js)に固定されており、ブラウザ目視確認は未実施。Mac既定ブラウザで手動確認を依頼中。

## 2026-06-15 第2版（国・割りもの・ラム対応）

変更前バックアップ：git tag `backup-before-v2`（HEAD=18201e3）

### 変更内容
- **国名を大ボタン3つに**：アメリカ🇺🇸 / オーストラリア🇦🇺 / 日本🇯🇵
- **日本選択時に「お名前」入力欄**を表示（会計データに guestName として保存・一覧に「（名前）」表示）
- **「その他の国」入力欄**を追加（自由入力。入力すると大ボタンの選択は解除）
- **ラム 800円**をドリンク既定に追加（既存データにも自動で補完）
- **ソーダを飲み物と分離**：ドリンク側の「ソーダ+100」トグルを廃止
- **割りもの（わりもの）機能を新設**：ソーダ・コーラ・トニックを既定登録（各100円）。会計画面に専用の「＋」ボタン群を設置し、割る時に押すと独立した明細として加算される
- **会計画面から直接「＋飲み物を追加」「＋割りものを追加」**できるインラインフォームを設置（登録すると即その注文に追加）
- ドリンクタブに「割りもの登録」セクションを追加（名前・値段の編集／削除）。「ソーダ追加料金」単一欄は廃止（割りもの個別価格に統合）
- 注文明細は「ドリンク＋割りもの」を一覧表示（割りものは「割」タグ付き）

### データ移行
- 旧データを読み込んだ場合：mixers が無ければ既定を補完、ラムが無ければ追加。旧 soda フラグ付き明細も一覧表示は互換維持。

### 検証
- 構文チェック（node --check）：OK
- ロジックテスト：ラム登録／割りもの既定／チャージ＋ドリンク＋割りもの合計／日本の名前保存／国別集計 すべて期待値一致
  - 例：日本・田中・チャージ＋ラム＋ソーダ割 = ¥1,900／アメリカ2名・ビール2＋コーラ割 = ¥3,700／売上 ¥5,600
